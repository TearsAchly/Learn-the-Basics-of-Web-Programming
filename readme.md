## Course Overview

This course explains the basics of web development, including how websites work and how to build and style web pages using HTML and CSS.

---

## Topics Covered

### 1. Introduction (40 minutes)
- Website fundamentals
- Server and client concepts
- Website anatomy
- Tools needed for web development

### 2. HTML Introduction (1 hour 10 minutes)
- Understanding HTML
- Basic HTML document structure

### 3. HTML Deep Dive (6 hours 45 minutes)
- Various HTML elements
- Organizing content using semantic HTML
- Attributes and generic elements
- Tables
- Inline vs block elements

### 4. CSS Introduction (1 hour 50 minutes)
- The role of CSS in web design
- How CSS works
- Writing CSS rules and syntax

### 5. CSS Deep Dive (15 hours 20 minutes)
- Selectors
- Font and text styling
- Foreground and background colors
- Box model
- Shadows
- Positioning
- Layout with float
- Media queries

### 6. Responsive Layout with Flexbox (4 hours 5 minutes)
- Building responsive layouts using Flexbox
- Combining Flexbox with media queries for better mobile experience

### 7. Final Project & Closing (10 hours 15 minutes)
- Implementing everything learned
- Improving and improvising a simple website project
